# Hero Video Widget - Usage Guide

## Overview
The Hero Video widget creates a full-screen video background hero section matching the Trip Kailash home page design.

## Features
- **Video Background**: Upload video directly from WordPress Media Library (supports MP4, WebM)
- **Trishul Icon**: Optional animated trident icon at the top
- **Large Heading**: Main hero title (e.g., "Walk the Path of Gods")
- **Subheading**: Descriptive text below the heading
- **Two CTA Buttons**: Primary (gold) and secondary (outlined) buttons
- **Scroll Indicator**: Animated "DISCOVER SACRED JOURNEYS" text with down arrow
- **Sound Toggle**: Mute/unmute button for video audio
- **Responsive**: Fully responsive design for all screen sizes

## How to Use in Elementor

### 1. Add the Widget
- Open your page in Elementor
- Search for "Hero Video" in the widgets panel
- Drag it to your page

### 2. Configure Content

#### Video Source
- Click "Choose Video" to upload from WordPress Media Library
- Recommended formats: MP4 (best compatibility), WebM
- Recommended resolution: 1920x1080 or higher
- Keep file size under 50MB for best performance

#### Trishul Icon
- Toggle "Show Trishul Icon" on/off
- Icon appears at the top center with floating animation

#### Heading & Subheading
- **Heading**: Main title (default: "Walk the Path of Gods")
- **Subheading**: Descriptive text (default: "Sacred luxury pilgrimages...")

#### CTA Buttons
- **Primary Button**: Gold button (default: "Begin Your Yatra")
- **Secondary Button**: Outlined button (default: "Speak with Tirtha Purohit")
- Set text and links for each button

#### Scroll Indicator
- Toggle "Show Scroll Indicator" on/off
- Customize the text (default: "DISCOVER SACRED JOURNEYS")
- Animated down arrow appears below text

### 3. Style Settings
- **Overlay Opacity**: Adjust the darkness of the overlay (0-100%)
- Default: 40% for good text readability

## Design Details

### Colors
- Primary Button: Gold (#b8860b)
- Secondary Button: White outline
- Text: White with subtle shadow for readability
- Trishul Icon: Gold with floating animation

### Typography
- Heading: Large serif font (Cinzel)
- Subheading: Clean sans-serif (Inter)
- Buttons: Bold, uppercase styling

### Animations
- Trishul icon floats up and down
- Scroll arrow bounces
- Buttons lift on hover
- Video scales slightly on card hover

## Tips for Best Results

1. **Video Selection**
   - Use high-quality footage of mountains, temples, or landscapes
   - Ensure good lighting and stable footage
   - Avoid videos with too much motion (can be distracting)

2. **Text Readability**
   - Adjust overlay opacity if text is hard to read
   - Keep heading concise (3-6 words ideal)
   - Subheading should be 1-2 lines

3. **Performance**
   - Compress videos before uploading
   - Use MP4 format for best browser compatibility
   - Consider using a poster image for slow connections

4. **Accessibility**
   - Ensure sufficient contrast between text and background
   - Provide meaningful button text
   - Video should not autoplay with sound (muted by default)
